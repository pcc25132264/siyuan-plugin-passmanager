# 思源笔记密码管理器插件

这是一款专业级的思源笔记密码管理插件，支持 AES-256 加密、密码生成以及自动锁定机制。

## 功能特性
- **安全性**: 采用 PBKDF2 密钥派生和 AES-256-GCM 加密，主密码不被保存。
- **组织管理**: 支持按分组管理密码。
- **便捷性**: 包含自定义密码生成器以及一键复制用户名/密码功能。
- **隐私保护**: 闲置 5 分钟或窗口失焦时自动锁定密码库。
- **数据隔离**: 您的加密数据被安全隔离存储于 `data/storage/petal/siyuan-plugin-passmanager` 目录下。

## 安装说明
1. 打开思源笔记 设置 > 插件 > 集市。
2. 搜索 “密码管理器” 并安装。
3. 启用该插件。

## 开发指南
- 核心加密由 `CryptoManager` (基于 Web Crypto API) 负责。
- 自动锁定由 `AutoLockManager` 监听鼠标及键盘事件。
- 测试用例请参阅 `tests/test_crypto.js`。
