# 思源密码管理器插件 (Siyuan Password Manager Plugin)

一款专业级的思源笔记密码管理器插件，采用行业标准加密技术、安全密码生成、加密文本管理和智能自动锁定机制，全面保护您的敏感信息。

## 开发初衷

这款插件的想法源于一个真实的安全隐患：思源笔记的 Git 同步插件会将 `data` 目录中的敏感信息提交到 GitHub，这可能导致隐私数据暴露于安全风险之中。为了解决这个问题，我开发了这款具有强加密功能的密码管理器插件，确保敏感凭证和私密文本得到妥善保护，并与普通笔记数据隔离，防止通过版本控制系统意外泄露。

## 核心功能

- **高级安全加密**：使用 AES-256-GCM 加密和 PBKDF2 密钥派生，提供最高级别的安全保护。
- **双重数据管理**：支持“密码列表”和“加密文本”双标签页无缝切换管理。
- **分类管理**：基于分组的密码存储，内置 5 个默认分类（工作、金融、社交、生活、未分类）并支持自定义分类。
- **用户友好界面**：直观的设计，支持字段快速复制按钮、密码显示/隐藏切换、全文搜索和列表排序。
- **智能自动锁定**：空闲或窗口失焦时自动锁定，防止未经授权的访问。
- **随机密码生成器**：添加密码时可快速创建包含数字、符号、大小写字母的强密码。
- **导入与导出**：支持将所有数据导出为 JSON 备份；同时支持导出为“未加密”或“加密”的 Markdown 笔记，直接保存在思源工作空间内。
- **透明的解密机制**：在插件设置中可以直接查看加密盐值（Salt）以及 Node.js 的本地手动解密代码示例，确保您在任何情况下都能掌握并访问自己的数据。
- **主密码找回（可选）**：支持在初始化时绑定思源笔记登录密码，作为忘记主密码时的备用恢复手段。
- **跨平台兼容**：完美适配思源笔记桌面端与移动端 UI。

## 安装

### 从插件市场安装
1. 打开思源笔记
2. 进入 **设置 > 插件 > 插件市场**
3. 搜索 "Password Manager" 或 "密码管理器"
4. 点击 **安装**，然后 **启用** 插件

### 手动安装
1. 从插件仓库下载最新版本发布包
2. 解压文件到 `data/plugins/siyuan-plugin-passmanager` 目录
3. 重启思源笔记
4. 进入 **设置 > 插件** 并启用本插件

## 使用方法

### 初始设置
1. 启用插件后，点击右上角顶栏的“锁”图标（或使用快捷键 `Shift+Cmd+P` / `Shift+Ctrl+P`）。
2. 设置一个主密码（强烈建议牢记，并可选输入思源登录密码作为找回凭证）。
3. 确认主密码，初始化安全加密库。

### 添加密码
1. 解锁并打开密码箱，确保处于“密码列表”标签页。
2. 点击“添加密码”按钮。
3. 输入标题、用户名、密码、网址并选择一个分类。
4. 你可以点击“生成”按钮快速创建一个强密码。
5. 点击“保存”安全存储。

### 管理加密文本
1. 切换到“加密文本”标签页。
2. 点击“添加文本”按钮。
3. 输入标题，选择分类，并在多行文本框中填入需要加密的敏感内容。
4. 点击“保存”，文本内容将会被安全加密存储。

### 导出数据
1. **导出为 JSON**：将整个密码库导出为一个本地 JSON 文件，方便备份。
2. **导出未加密笔记**：将密码和加密文本以明文 Markdown 格式导出到思源名为“密码箱”的笔记本中。
3. **导出加密笔记**：同样导出到思源笔记中，但会用实际的加密字符串（密文）替换原本的明文密码和文本内容。

### 设置与自动锁定
- 通过思源笔记自带的插件设置面板，可以调整本插件的相关设置。
- 你可以开启/关闭“总是要求解锁”。如果关闭此选项，主密码会保存在本地（安全性降低），以便下次打开免密解锁。
- 支持在设置面板中直接“修改主密码”。
- **手动解密设置**：在插件设置中，你可以查看加密算法的详细信息、唯一的加密盐值（Salt），以及一段 Node.js 的代码示例。这段代码展示了如果脱离了插件，如何使用主密码手动解密 `vault-data.json` 文件。

## 安全说明

- **主密码**：您的主密码不会以明文形式存储在任何地方（除非你主动关闭了“总是要求解锁”选项）——它仅用于派生加密密钥。
- **加密算法**：所有的密码字段和加密文本内容，均使用 AES-256-GCM 军用级加密标准进行处理。
- **数据存储**：加密数据安全保存在您本地的 `data/storage/petal/siyuan-plugin-passmanager/vault-data.json` 文件中。
- **隐私保护**：插件运行在本地，不会向任何外部服务器发送数据。
- **数据备份**：请定期备份您的思源笔记 `data` 目录以防止数据丢失。

## 故障排除

### 忘记主密码
- 如果您忘记了主密码，且在初始设置时没有绑定思源笔记密码用于恢复，那么**没有任何办法**可以恢复您的加密数据。
- 请务必记住您的主密码或将其保存在安全的地方。

### 插件无法加载
- 确保您使用的是较新版本的思源笔记（建议 v3.0.0+）。
- 检查开发者控制台是否有任何报错信息。
- 尝试禁用后重新启用，或重新安装插件。
