# 思源密码管理器插件

一款专业级的思源笔记密码管理器插件，采用行业标准加密技术、安全密码生成和智能自动锁定机制，全面保护您的敏感信息。

> **⚠️ 免责声明**：本插件按"原样"提供，不提供任何担保。开发者和贡献者**不对**因使用本插件而导致的任何数据丢失、安全漏洞或损害承担责任。用户需自行负责维护数据备份并确保主密码的安全。使用风险自负。

## 核心功能

- **高级安全**：使用 AES-256-GCM 加密和 PBKDF2 密钥派生，提供最高级别的安全保护
- **分类管理**：基于分组的密码存储，便于组织和快速访问
- **用户友好界面**：直观的设计，支持快速复制按钮和搜索功能
- **智能自动锁定**：空闲或窗口失焦时自动锁定，防止未经授权的访问
- **安全数据存储**：加密数据安全存储在 `data/storage/petal/siyuan-plugin-passmanager`
- **随机密码生成器**：创建强大且独特的密码，支持自定义参数
- **跨平台兼容**：在所有思源笔记支持的平台上无缝运行

## 安装

### 从插件市场安装
1. 打开思源笔记
2. 进入 **设置 > 插件 > 插件市场**
3. 搜索 "Password Manager" 或 "密码管理器"
4. 点击 **安装**，然后 **启用** 插件

### 手动安装
1. 从插件仓库下载最新版本
2. 解压文件到 `data/plugins/siyuan-plugin-passmanager`
3. 重启思源笔记
4. 进入 **设置 > 插件** 并启用插件

## 使用方法

### 初始设置
1. 启用插件后，点击侧边栏中的密码管理器图标
2. 设置主密码（请记住此密码，无法恢复）
3. 确认您的主密码

### 添加密码
1. 点击 "添加" 按钮
2. 输入网站/应用名称、用户名、密码
3. 可选择添加备注并分配到分组
4. 点击 "保存" 安全存储密码

### 生成密码
1. 点击 "生成" 按钮
2. 调整密码长度和字符类型
3. 点击 "复制" 将生成的密码复制到剪贴板

### 自动锁定设置
- 插件在 5 分钟无操作后自动锁定
- 当思源笔记窗口失去焦点时也会锁定
- 您可以在插件设置中调整自动锁定超时时间

## 开发

### 项目结构
- `src/crypto.js`：处理 Web Crypto API 的加密/解密
- `src/autolock.js`：管理空闲状态检测和自动锁定功能
- `src/storage.js`：处理安全数据存储和检索
- `src/ui.js`：实现用户界面组件
- `tests/test_crypto.js`：加密功能的测试用例

### 构建插件
1. 克隆仓库
2. 运行 `npm install` 安装依赖
3. 运行 `npm run build` 构建插件
4. 构建后的文件将在 `dist` 目录中

## 安全说明

- **主密码**：您的主密码不会存储在任何地方 - 仅用于派生加密密钥
- **加密**：所有密码使用 AES-256-GCM 加密，这是军用级加密标准
- **数据存储**：加密数据本地存储在您的思源笔记数据目录中
- **隐私**：不会向任何外部服务器发送数据
- **备份**：定期备份您的思源笔记数据目录以防止数据丢失

## 故障排除

### 忘记主密码
- 如果您忘记了主密码，将无法恢复您的加密数据
- 请务必记住您的主密码或将其保存在安全的地方

### 插件无法加载
- 确保您使用的是最新版本的思源笔记
- 检查浏览器控制台是否有任何错误信息
- 尝试重新安装插件
